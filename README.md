# hola-mundo-express
## Local
Clona el proyecto 

~~~bash  
  git clone (ssh - link-to-proyect)
~~~

Abre el proyecto 

~~~bash  
  cd project
~~~

Instala las dependencias  

~~~bash  
npm install express

npm install nodemon
~~~

Inicia el servidor *(por defecto en puerto 3000)*

~~~bash  
npm run dev
~~~
